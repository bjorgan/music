# music
Generate sounds using SFML.

Compile using:

1. `mkdir build`
2. `cd build`
3. `cmake ..`
4. `make`

Run using:

`./music`
Press any key.

`./music keyboard`
Use z-m and s-j:

```
  s d   g h j
 z x c v b n m
(C D E F G A H)
```

`./music chords`
Play chords in sequence, same setup as above. Use uppercase letters for minor chords.
